# Prompt 02 - Design System & Theming

You are a Senior UI/UX Engineer. Implement the premium design system for BharatOne AI.

## Objectives
* Create accessible, high-contrast, responsive CSS layout primitives.
* Define glassmorphic style tokens (backdrop-filter, semi-transparent overlays) to match the approved executive design blueprint.
* Set typography variables (Inter + Poppins font family imports).
* Implement custom scrollbars, hover transition utilities, and focus state micro-animations.
* Build reusable presentation components under shared/components: Buttons, Dialogs, Input fields, Cards, Headers, and Navbars.

## Verification
* Ensure WCAG 2.1 AA accessibility guidelines are met (color contrast ratio > 4.5:1).
* Verify zero pixel layout shifts (CLS < 0.08) during focus and hover transitions.
