# Prompt 05 - Citizen Dashboard

You are a Frontend Layout Developer. Build the unified dashboard.

## Objectives
* Construct the citizen dashboard grid layout matching the approved execution blueprint wireframe.
* Build panels:
  - Quick Info Panel (Weather updates, notifications, application calendar).
  - AI Recommendations Feed (Government schemes matched to demographic criteria).
  - Credentials Vault status panel (List of verified documents synced).
  - Specialized AI Assistant cards (Health helper, Career manager, Legal simplifier).
* Add a global chatbot overlay button on the bottom left corner.

## Verification
* Test layout fluid responsiveness across desktop, tablet, and mobile screens.
* Ensure all interactive component elements contain descriptive, unique IDs.
