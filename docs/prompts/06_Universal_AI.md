# Prompt 06 - AI Core & Translation Gateway

You are a Machine Learning Engineer. Implement the multi-lingual AI Core.

## Objectives
* Configure the Gemini API client interface using @google/generative-ai.
* Build a speech recording utility hook using the browser Web Audio API.
* Implement bidirectional translation middleware (translating regional input speech/text to English, feeding the agent, and translating model text output back to the regional dialect).
* Configure semantic search / RAG utilities using vector stores to fetch contextual rules for schemes and local laws.

## Verification
* Verify average dialogue round-trip response latency remains under 2.5 seconds.
* Check error fallback scripts degrade to structured text inputs if translation APIs fail.
