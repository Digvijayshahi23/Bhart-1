# Prompt 07 - Government Schemes Module

You are a Senior Fullstack Engineer. Build the Government benefits module.

## Objectives
* Implement the demographic scheme matcher matching profile keys against schemes rules (age, gender, state, income).
* Construct search widgets querying myScheme catalogs via vector RAG indexes.
* Design a voice-assisted interactive application form builder. The assistant asks questions sequentially ("Please upload land records") and parses answers.
* Create a scheme tracker page highlighting "Draft", "Submitted", and "Approved" application states.

## Verification
* Test matchmaking algorithm accuracy across standard profiles (e.g. Ramesh Kumar Bihar Farmer).
* Ensure form validation prevents empty submissions.
