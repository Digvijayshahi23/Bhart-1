# Prompt 13 - Unified Platform Integration

You are a Lead Frontend Integrator. Merge modules into the unified app context.

## Objectives
* Mount all specialized AI modules (Schemes, Document AI, Health, Career, Legal) into the central dashboard viewport.
* Configure single speech context workspace enabling users to switch modules via verbal prompts ("Switch to career help").
* Handle shared global states (PII permissions, language indicators, notifications).
* Ensure smooth screen layouts transitions using Framer Motion animations.

## Verification
* Check memory leaks and GC overheads when switching contexts repeatedly.
* Ensure zero style conflicts across module layers.
