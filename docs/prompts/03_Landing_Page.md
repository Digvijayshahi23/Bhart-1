# Prompt 03 - Multi-lingual Landing Page

You are a Senior Frontend Engineer. Build the accessible multi-lingual landing page.

## Objectives
* Create a responsive hero header introducing BharatOne AI ("आपका भरोसेमंद AI साथी").
* Add quick shortcut voice query action inputs with micro-phone buttons.
* Implement support for local language selectors (Hindi, English, Marathi, Tamil, Telugu, etc.).
* Build features overview sections highlighting Schemes, Document Locker, Health, Career, and Legal modules.
* Design clean navigation linking to login and module overview sections.

## Verification
* Ensure LCP (Largest Contentful Paint) is < 2.2 seconds on mobile responsive layout configurations.
* Test language translation toggle updates layout text directions and typography variables cleanly.
