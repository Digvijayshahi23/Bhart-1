# Prompt 10 - Career AI & Resume Builder

You are a Product Engineer. Build the Career assistance module.

## Objectives
* Build a voice-guided Resume wizard generating professional structured PDF files using jsPDF.
* Implement job board matching queried skills against National Career Service (NCS) API vacancies.
* Construct skill gap analyzer suggesting training courses (PMKVY, SWAYAM).
* Design a conversational mock interview coach conducting voice-based practice sessions and scoring pronunciations/grammar.

## Verification
* Test PDF layout rendering compiles cleanly without text clipping.
* Check job recommendation matching speed.
